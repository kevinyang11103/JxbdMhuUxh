# 水果农产品销售商城系统 python385

## 项目概述

水果农产品销售商城系统是一个基于Python技术的全功能在线销售平台，旨在提供高效、稳定的水果农产品交易解决方案。

## 技术栈

- 后端：Python 3.8、Django
- 前端：Vue 2
- 数据库：MySQL
- 开发环境：PyCharm

## 功能模块

系统主要包括以下功能模块：

- 用户注册与登录
- 商品浏览与搜索
- 购物车管理
- 订单处理
- 用户与管理员权限管理

## 系统角色

- 管理员：负责商品管理、订单管理、用户管理等后台操作。
- 用户：可以在商城浏览商品、添加至购物车、下单购买等。

## 运行环境

- 后端：需配置Python 3.8及以上版本，安装Django框架。
- 前端：确保Node版本高于14，以便配置cnpm环境。
- 数据库：安装MySQL数据库。

## 部署步骤

1. 安装配置Python环境。
2. 安装配置MySQL数据库。
3. 部署后端Django服务。
4. 部署前端Vue项目。

## 目录结构

```
项目根目录/
├── backend    // 后端Django代码
├── frontend   // 前端Vue代码
└── docs        // 项目文档与图表模板
```
